Detailed Description

The settings.py file is used to store configuration settings using dataclasses. 
It includes 2 dataclasses and a function to create a settings object and read it from the input file.
The settings.py file serves as a module for managing settings and configurations for bots. 
It utilizes the environs library to read settings from an environment file and stores them 
in data classes.

◆ get_settings()

settings.get_settings ( str path) 
This function retrieves settings from a specified path (presumably a settings file). It reads environment variables from that file and constructs a Settings object containing information related to bots (such as bot tokens and admin IDs).

Parameters:
 - path: A string representing the path to the settings file.
Returns: The constructed Settings object.
