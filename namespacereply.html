 get_reply_keyboard ()
 

Detailed Description

Contains a function for creating a custom reply keyboard. It provides the ability to generate a user-friendly 
keyboard for responses in Telegram. The get_reply_keyboard() function creates a keyboard with buttons for file loading, 
help invocation, and file selection.

reply.get_reply_keyboard ( ) 
This function generates a custom reply keyboard for use in a chat interface.
The resulting keyboard can be used to provide user-friendly options for interacting with the bot.

Behavior:
 - Creates a ReplyKeyboardBuilder instance.
 - Adds three buttons to the keyboard: “Load File,” “Help,” and “Select File.”
 - Adjusts the layout of the keyboard (e.g., spacing, alignment).
 - Returns the generated reply keyboard markup with specified properties (resize keyboard, one-time keyboard, 
    and input field placeholder).
