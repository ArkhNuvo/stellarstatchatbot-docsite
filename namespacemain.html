StellarStatBot 1.0
The project is a statistical chatbot developed for the Telegram platform using the aiogram library. It has the capability of performing basic data analysis, including calculation of simple statistics (such as mean, median, standard deviation), and generation of univariate graphs based on user-uploaded data.

Detailed Description

The main.py file serves as the entry point for running the bot application. 
Here's a general overview of what it does:

    1. It imports necessary modules and libraries such as aiogram, asyncio, logging, and others.
    2. The start_bot and stop_bot functions send messages indicating the activation and deactivation 
    of the bot, respectively.
    3. The start function initializes the bot, sets up event handlers for various commands and messages, 
    and starts polling for updates.
    4. Event handlers are registered for commands like start, info, help, as well as for specific types 
    of messages like photos and documents.
    5. The event handlers execute corresponding functions when triggered by user actions or messages.
    6. The script runs the start function using asyncio.run() when executed directly.

Overall, main.py orchestrates the setup and operation of the bot, handling incoming messages and executing 
appropriate actions based on the defined event handlers.

main.start ( ) 
This asynchronous function initializes and starts the bot. It sets up event handlers for 
various commands and actions.

Behavior:
 - Configures logging to display log messages with timestamps and relevant information.
 - Creates an instance of the bot using the provided bot token and sets the parse mode to “HTML”.
 - Initializes a dispatcher (dp) to handle incoming messages and callbacks.
 - Registers event handlers for various commands and actions (e.g., startup, shutdown, photo messages, hello messages, file loading, etc.).
 - Starts polling for updates from the Telegram API.
 - Closes the bot session when the function exits (using a finally block).
