Detailed Description

The file stat.py contains functions for analyzing statistical data from CSV files. 
It allows importing data from a CSV file into a DataFrame object, displaying column information, 
generating descriptive summaries, and computing various statistical metrics such as mean, median, 
mode, range, etc. Additionally, the file implements functions for creating various types of plots, 
such as histograms, box plots, and others, for data visualization purposes.
