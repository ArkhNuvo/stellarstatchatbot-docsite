Detailed Description

This class represents general user data downloaded from Telegram. 
It includes information such as the userâ€™s name, ID, and folder path for uploaded files.

Attributes:
 - name: A string representing the name of the user. This name is used in personal bot messages.
 - id: A string representing the ID of the user. It is used to create a unique 
    folder path for storing files related to that user.
 - folder_path: A string representing the folder path associated with the user. 
    Each user has their own personal folder path.
 - file_count: An integer representing the count of files for the user. 
    It indicates the number of files the user has uploaded.
