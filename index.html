StellarStatBot. How to Use Guide

General

StellarStatBot is a tool designed for users who want to quickly get acquainted with their data. It offers a set of features to perform initial data analysis, including:

  • Calculating basic statistics: This includes measures such as mean, median, mode, standard deviation, quartiles, etc.
  • Determining data types: StellarStatBot can identify the data types of your variables, such as numeric, categorical.
  • Managing files: You can save and delete files from your personal storage within the tool.
  • Data visualization: StellarStatBot allows you to visualize your data in a variety of ways, including graphs, pie charts, histograms, etc.

With StellarStatBot, you can gain valuable insights into your data and make informed decisions based on your findings.

Key Features:

  • User-friendly interface: The tool is designed to be easy to use, even for beginners.
  • Comprehensive functionality: StellarStatBot offers a wide range of features for initial data analysis.
  • Flexibility: You can customize your analysis by selecting the features you need.
  • Portability: The tool is available as a web application, so you can access it from anywhere.

How To Use the Bot

Step I. Commands

At the beginning of working with the bot, you will see information about the service, as well as three commands in the "Menu" tab:

  • /start: Initiates interaction with the bot and serves as the starting point for each bot. Sometimes it is used to "reset the session" and start over. For convenience, you can regularly use this command when you want to update the bot or clear chat history.
  • /help: Provides guidance on how to use the bot.
  • /info: Displays information about the bot and outlines plans for future development of this project.

Step II. Reply Keyboard

Reply Keyboard is the primary tool for controlling the bot. It is available at any stage of analysis. It has two simple functions for working with files and a help button (which will be modified in the future). If desired, the keyboard can be hidden if it interferes with data exploration. When buttons are pressed, the keyboard disappears. To use its functionality again, click the square with four dots. Here's a more detailed description of the button functions:

  • "Load File": Signals that the bot is ready to accept a file.
  • "Help": Provides guidance on which files can be uploaded.
  • "Select File": Displays a list of previously uploaded files. If no files exist in the user's personal storage, a message will indicate that no files are available.

Step III. File Management:

Step III.1. Help Button:

The “Help” button serves as a reminder of the file formats that the bot processes.

Step III.2. Load Button:

After clicking the "Load File" button, the bot will indicate that it is ready to accept a file. It's important to remember that the processing time for the file depends on various factors (such as file size, service load, internet connection, etc.). If successful, the bot will confirm that the file has been selected and offer the following options:

  • Show Columns: This option displays the columns in your database with the format automatically determined by the bot.
  • Show Description: This option provides consolidated information about the file (including mean, median, unique value count, maximums, and minimums). The data is presented in the form of a table on a generated .png image.
  • Show Info: This option presents basic information about the dataset, including the number of non-null values, data format, and column names, all in a single message.
Step III.3. Select File Button:

Clicking the "Select File" button will redirect you to your personal storage if you have previously interacted with the bot. When choosing this option, you will have the following two choices:

  1. Select any file that you have previously uploaded. The bot will then redirect you to the option menu above the file: (Show Columns, Show Description, Show Info). Keep in mind that if, for any reason, the file did not upload successfully, it will not appear in your list. Please try uploading the file again.
  2. Delete the file from your storage. In this case, you will no longer be able to use it, and you will need to upload it again.

Step IV. Columns Menu

When you press the "Show Columns" button, the bot scans the received or selected .csv file using the pandas library and converts the dataset into a DataFrame. Next, it reads the columns in the dataset and creates inline buttons for each variable. Simultaneously, pandas determines the default data type of the column (in the future, a function to change the data type of the column will be added). When you click any button, you enter the column menu with a choice of analytical tools, starting from calculating the mean (for quantitative data) and counting the number of values (for qualitative data), and ending with Violin plots and Pie charts.

Step V.1 Numerical Variables options

When you select a quantitative variable, you will have access to the following options:

Descriptive Statistics:
  • Mean: The arithmetic average of all values in the column.
  • Median: The middle value that divides the dataset into two equal parts.
  • Mode: The most frequently occurring value.
  • Trimmed Mean: The mean excluding a certain percentage of outliers.
  • Range: The difference between the maximum and minimum values.
  • Interquartile Range (IQR): The difference between the 75th and 25th percentiles.
  • Variance: The average of squared deviations from the mean.
  • Standard Deviation: The square root of the variance.
  • Mean Absolute Deviation (MAD): The average of absolute deviations from the mean.
Graphs and Charts for the selected column:
  • Histogram: Visualizes the distribution of values in the column.
  • Box Plot: Displays the median, quartiles, and outliers.
  • Strip Plot: Shows each observation as a point.
  • Swarm Plot: Non-overlapping points, suitable for small datasets. (Note: Works only with data below 18 MB)
  • Violin Plot: Combines a histogram and a box plot.

Step V.2 Categorical Variable Options

When you select a categorical variable, the following options for descriptive statistics become available:

  • Count_unique (Count Unique Values): Shows the number of unique values in the dataset.
  • Unique (Show Unique Values): Lists all unique values. **Be aware that Telegram may hide a large number of unique values, or the message may become too large.**
  • Value Counts (Normalized): Displays the count of each unique value as a percentage of the total number of unique values.

The following charts are available for categorical variables:

  • Pie chart (Pie Chart): Visualizes the distribution of unique values as a pie chart.
  • Count bar plot (Bar Chart): Shows the count of each unique value. Currently, there is a limitation: if the number of unique values exceeds 10, only the top 5 most frequent ones will be shown, and the rest will be grouped into the "Others" category.
